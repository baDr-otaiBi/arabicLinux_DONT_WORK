- Dynamic Linking → الربط الديناميكي
- Virtual Memory → الذاكرة الافتراضية  
- Process Scheduling → جدولة العمليات
- Copy-on-Write → النسخ عند الكتابة
- Lazy Loading → التحميل الكسول
- Hot Reload → إعادة التحميل الساخن
- Sandboxing → العزل الأمني
- وغيرها المئات من المصطلحات
