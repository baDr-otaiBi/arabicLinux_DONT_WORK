## الدليل التعليمي لفهم توزيعات لينكس

### مقدمة

هذا الدليل التعليمي يشرح المفاهيم الأساسية لتوزيعات لينكس من خلال هذا المشروع التجريبي. الهدف هو تبسيط المفاهيم المعقدة وجعلها مفهومة للجميع.

---

## بنية نظام الملفات في لينكس

### ما هو نظام الملفات؟

نظام الملفات هو الطريقة التي ينظم بها نظام التشغيل الملفات والمجلدات على القرص الصلب. في لينكس، كل شيء هو ملف، حتى الأجهزة!

### المجلدات الرئيسية في توزيعتنا العربية

#### 1. مجلد `/نظام` (يقابل `/bin` و `/sbin` في لينكس التقليدي)

**الغرض:**
- يحتوي على البرامج الأساسية للنظام
- الملفات التنفيذية الضرورية للإقلاع والعمليات الأساسية

**في توزيعتنا:**
```
/نظام
├── معلومات_النظام.conf    # معلومات التوزيعة والإصدار
└── أساسي/                  # برامج النظام الأساسية
```

**ماذا نتعلم:**
- كيفية تخزين معلومات النظام
- تنظيم الملفات الأساسية
- قراءة إعدادات التوزيعة

---

#### 2. مجلد `/إقلاع` (يقابل `/boot` في لينكس التقليدي)

**الغرض:**
- يحتوي على ملفات بدء التشغيل
- النواة وبرنامج الإقلاع

**في توزيعتنا:**
```
/إقلاع
├── بدء_النظام.sh          # برنامج بدء التشغيل الرئيسي
└── برامج/                 # برامج الإقلاع الإضافية
```

**ماذا نتعلم:**
- عملية إقلاع النظام (Boot Process)
- كيفية تهيئة المكونات
- ترتيب تحميل الخدمات

**مراحل الإقلاع في توزيعتنا:**
1. **المرحلة الأولى**: عرض معلومات البدء
2. **المرحلة الثانية**: التحقق من بنية المجلدات
3. **المرحلة الثالثة**: تحميل معلومات النظام
4. **المرحلة الرابعة**: تهيئة النظام
5. **المرحلة الخامسة**: إتمام الإقلاع

---

#### 3. مجلد `/تطبيقات` (يقابل `/usr/bin` في لينكس التقليدي)

**الغرض:**
- يحتوي على التطبيقات والبرامج المثبتة
- البرامج التي يستخدمها المستخدمون

**في توزيعتنا:**
```
/تطبيقات
├── إدارة_الحزم.sh          # نظام إدارة الحزم
├── إدارة_المستخدمين.sh     # نظام إدارة المستخدمين
├── نظام/                   # تطبيقات النظام
└── مستخدم/                 # تطبيقات المستخدم
```

**ماذا نتعلم:**
- كيفية تنظيم التطبيقات
- إدارة الحزم والبرامج
- التفاعل مع البرامج النصية

---

#### 4. مجلد `/إعدادات` (يقابل `/etc` في لينكس التقليدي)

**الغرض:**
- يحتوي على ملفات الإعدادات
- التكوينات العامة للنظام والبرامج

**في توزيعتنا:**
```
/إعدادات
├── إعدادات_النظام.conf    # الإعدادات الرئيسية
├── نظام/                   # إعدادات النظام
└── شبكة/                   # إعدادات الشبكة
```

**ماذا نتعلم:**
- كيفية تخزين الإعدادات
- تنسيق ملفات الإعدادات
- تطبيق التكوينات

---

#### 5. مجلد `/مستخدم` (يقابل `/home` في لينكس التقليدي)

**الغرض:**
- يحتوي على مجلدات المستخدمين الشخصية
- الملفات والإعدادات الخاصة بكل مستخدم

**في توزيعتنا:**
```
/مستخدم
├── جذر/                    # مجلد المستخدم الجذر
└── عام/                    # مجلد عام للجميع
```

**ماذا نتعلم:**
- فصل بيانات المستخدمين
- الصلاحيات والخصوصية
- إدارة المستخدمين

---

#### 6. مجلد `/مكتبات` (يقابل `/lib` في لينكس التقليدي)

**الغرض:**
- يحتوي على المكتبات المشتركة
- الموارد المستخدمة من البرامج المختلفة

**في توزيعتنا:**
```
/مكتبات
└── مشتركة/                # مكتبات مشتركة بين البرامج
```

**ماذا نتعلم:**
- مفهوم المكتبات المشتركة
- إعادة استخدام الكود
- تقليل حجم البرامج

---

#### 7. مجلد `/أجهزة` (يقابل `/dev` في لينكس التقليدي)

**الغرض:**
- يحتوي على معلومات الأجهزة
- واجهات التعامل مع العتاد

**في توزيعتنا:**
```
/أجهزة
└── معلومات/               # معلومات الأجهزة المتصلة
```

**ماذا نتعلم:**
- كيفية تمثيل الأجهزة كملفات
- التواصل مع العتاد
- كشف الأجهزة

---

## عملية الإقلاع

### ما هو الإقلاع؟

الإقلاع هو عملية بدء تشغيل نظام التشغيل عند تشغيل الكمبيوتر. تتضمن هذه العملية عدة مراحل:

### مراحل الإقلاع التفصيلية

#### المرحلة الأولى: فحص الأجهزة الأساسية
- فحص العتاد
- تحميل برنامج الإقلاع الأولي

#### المرحلة الثانية: محمل الإقلاع
- تحميل النواة
- تمرير المعاملات

#### المرحلة الثالثة: النواة
- تهيئة العتاد
- تحميل برامج التشغيل

#### المرحلة الرابعة: نظام التهيئة
- بدء الخدمات الأساسية
- تهيئة النظام

#### المرحلة الخامسة: تسجيل الدخول
- عرض شاشة تسجيل الدخول
- بدء واجهة المستخدم

## إدارة الحزم

### ما هي الحزم؟

الحزم هي برامج معبأة بطريقة منظمة تسهل تثبيتها وإزالتها. كل حزمة تحتوي على:

1. **الملفات التنفيذية**: البرنامج نفسه
2. **المكتبات**: المكونات المساعدة
3. **الإعدادات**: ملفات التكوين
4. **البيانات الوصفية**: معلومات عن الحزمة

### مدير الحزم في توزيعتنا

برنامج `إدارة_الحزم.sh` يوفر:

- **التثبيت**: إضافة حزم جديدة
- **الإزالة**: حذف حزم موجودة
- **البحث**: إيجاد حزم محددة
- **القائمة**: عرض الحزم المثبتة
- **التحديث**: تحديث قاعدة البيانات

### كيف يعمل مدير الحزم؟

```bash
# 1. المستخدم يطلب تثبيت حزمة
./تطبيقات/إدارة_الحزم.sh تثبيت حزمة_تجريبية

# 2. المدير يبحث عن الحزمة في المستودع

# 3. يتحقق من التبعيات (Dependencies)

# 4. يقوم بتحميل الحزمة

# 5. يفك الضغط ويثبت الملفات

# 6. يسجل الحزمة في قاعدة البيانات
```

## إدارة المستخدمين

### لماذا نحتاج لعدة مستخدمين؟

1. **الأمان**: فصل الصلاحيات
2. **الخصوصية**: بيانات مستقلة لكل مستخدم
3. **التنظيم**: إدارة أفضل للموارد

### نظام إدارة المستخدمين

برنامج `إدارة_المستخدمين.sh` يوفر:

- **إضافة مستخدمين جدد**
- **حذف مستخدمين**
- **عرض معلومات المستخدم**
- **قائمة جميع المستخدمين**

### المعلومات المخزنة لكل مستخدم

```bash
# ملف معلومات المستخدم يحتوي على:
اسم_المستخدم="أحمد"          # اسم المستخدم
تاريخ_الإنشاء="2026-01-27"   # متى تم إنشاء الحساب
المجموعة="مستخدمون"          # المجموعة التي ينتمي إليها
الصلاحيات="عادي"             # مستوى الصلاحيات
الحالة="نشط"                 # حالة الحساب
```

## الصلاحيات والأمان

### أنواع الصلاحيات

1. **قراءة (Read)**: عرض المحتوى
2. **كتابة (Write)**: تعديل المحتوى
3. **تنفيذ (Execute)**: تشغيل البرامج

### المستخدم الجذر (Root)

- أعلى صلاحيات في النظام
- يمكنه القيام بأي شيء
- يجب استخدامه بحذر

### المستخدمون العاديون

- صلاحيات محدودة
- لا يمكنهم تعديل ملفات النظام
- أكثر أماناً للاستخدام اليومي

## إعدادات الشبكة

### المفاهيم الأساسية

#### عنوان IP
- عنوان فريد لكل جهاز على الشبكة
- مثال: 192.168.1.100

#### قناع الشبكة (Subnet Mask)
- يحدد نطاق الشبكة
- مثال: 255.255.255.0

#### البوابة (Gateway)
- نقطة الخروج من الشبكة المحلية
- مثال: 192.168.1.1

#### DNS
- يترجم أسماء النطاقات إلى عناوين IP
- مثال: google.com → 142.250.185.46

## المراقبة والسجلات

### أهمية السجلات (Logs)

السجلات تسجل:
- الأحداث المهمة في النظام
- الأخطاء والمشاكل
- محاولات تسجيل الدخول
- أنشطة المستخدمين

### مستويات السجلات

1. **تفصيلي (Verbose)**: كل شيء
2. **عادي (Normal)**: الأحداث المهمة
3. **أخطاء فقط**: المشاكل فقط

## تمارين تعليمية

### تمرين 1: استكشاف بنية الملفات

```bash
# قم باستكشاف كل مجلد وفهم محتوياته
ls -la نظام/
ls -la إقلاع/
ls -la تطبيقات/
ls -la إعدادات/
```

### تمرين 2: تشغيل النظام

```bash
# قم بتشغيل نظام الإقلاع وراقب المراحل المختلفة
./إقلاع/بدء_النظام.sh
```

### تمرين 3: إدارة الحزم

```bash
# جرب تثبيت وإزالة حزمة
./تطبيقات/إدارة_الحزم.sh تثبيت حزمة_تجريبية
./تطبيقات/إدارة_الحزم.sh قائمة
./تطبيقات/إدارة_الحزم.sh إزالة حزمة_تجريبية
```

### تمرين 4: إدارة المستخدمين

```bash
# أنشئ مستخدماً جديداً واستكشف معلوماته
./تطبيقات/إدارة_المستخدمين.sh إضافة محمد
./تطبيقات/إدارة_المستخدمين.sh معلومات محمد
./تطبيقات/إدارة_المستخدمين.sh قائمة
```

### تمرين 5: قراءة الإعدادات

```bash
# اقرأ ملف الإعدادات وافهم كل خيار
cat إعدادات/إعدادات_النظام.conf
```

## نصائح للتعلم

1. **اقرأ التعليقات**: كل ملف يحتوي على شرح مفصل
2. **جرب الأوامر**: لا تخف من التجربة
3. **افهم قبل الحفظ**: افهم لماذا وكيف
4. **قارن مع لينكس الحقيقي**: ابحث عن المقابلات
5. **اسأل واستكشف**: الفضول هو أفضل معلم
