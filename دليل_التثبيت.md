## دليل تثبيت واستخدام توزيعة لينكس العربية

### مقدمة

هذا دليل شامل لتثبيت واستخدام توزيعة لينكس العربية التجريبية. يهدف هذا الدليل إلى تسهيل عملية التثبيت والإعداد للمستخدمين الجدد.

---

## المتطلبات الأساسية

### المتطلبات التقنية

1. **نظام التشغيل الأساسي**
   - أي توزيعة لينكس حديثة
   - أو نظام يدعم Bash shell

2. **المتطلبات البرمجية**
   - Bash 4.0 أو أحدث
   - دعم UTF-8
   - Git (لاستنساخ المستودع)

3. **الذاكرة والمساحة**
   - ذاكرة عشوائية: 512 ميجابايت على الأقل
   - مساحة تخزين: 100 ميجابايت على الأقل

### المتطلبات اللغوية

- دعم اللغة العربية في الطرفية
- خطوط عربية مثبتة في النظام
- دعم الاتجاه من اليمين إلى اليسار (RTL)

---

## التحميل والتثبيت

### الخطوة الأولى: استنساخ المستودع

افتح الطرفية وأدخل الأوامر التالية:

```bash
# الانتقال إلى المجلد الرئيسي
cd ~

# استنساخ المستودع
git clone https://github.com/baDr-otaiBi/-

# الدخول إلى مجلد المشروع
cd -
```

### الخطوة الثانية: منح صلاحيات التنفيذ

```bash
# منح صلاحيات التنفيذ لجميع البرامج النصية
chmod +x إقلاع/بدء_النظام.sh
chmod +x تطبيقات/إدارة_الحزم.sh
chmod +x تطبيقات/إدارة_المستخدمين.sh
```

### الخطوة الثالثة: تشغيل نظام الإقلاع

```bash
# تشغيل برنامج بدء النظام
./إقلاع/بدء_النظام.sh
```

## الاستخدام الأساسي

### تشغيل النظام

بعد التثبيت، يمكنك تشغيل النظام في أي وقت:

```bash
cd ~/- 
./إقلاع/بدء_النظام.sh
```

### إدارة الحزم

استخدم برنامج إدارة الحزم لتثبيت وإزالة البرامج:

```bash
# عرض المساعدة
./تطبيقات/إدارة_الحزم.sh مساعدة

# تثبيت حزمة
./تطبيقات/إدارة_الحزم.sh تثبيت اسم_الحزمة

# إزالة حزمة
./تطبيقات/إدارة_الحزم.sh إزالة اسم_الحزمة

# عرض الحزم المثبتة
./تطبيقات/إدارة_الحزم.sh قائمة

# البحث عن حزمة
./تطبيقات/إدارة_الحزم.sh بحث اسم_الحزمة
```

### إدارة المستخدمين

استخدم برنامج إدارة المستخدمين لإضافة وحذف المستخدمين:

```bash
# عرض المساعدة
./تطبيقات/إدارة_المستخدمين.sh مساعدة

# إضافة مستخدم جديد
./تطبيقات/إدارة_المستخدمين.sh إضافة أحمد

# عرض معلومات مستخدم
./تطبيقات/إدارة_المستخدمين.sh معلومات أحمد

# حذف مستخدم
./تطبيقات/إدارة_المستخدمين.sh حذف أحمد

# عرض قائمة المستخدمين
./تطبيقات/إدارة_المستخدمين.sh قائمة
```

## الإعدادات

### تعديل إعدادات النظام

يمكنك تعديل إعدادات النظام من خلال ملف الإعدادات:

```bash
# فتح ملف الإعدادات للتحرير
nano إعدادات/إعدادات_النظام.conf
```

### الإعدادات المتاحة

- **إعدادات اللغة**: تغيير اللغة والترميز
- **إعدادات الشبكة**: تكوين الشبكة
- **إعدادات الأمان**: ضبط مستوى الأمان
- **إعدادات الأداء**: تحسين الأداء

## هيكل المجلدات

### `/نظام` - ملفات النظام الأساسية
يحتوي على:
- `معلومات_النظام.conf`: معلومات التوزيعة

### `/إقلاع` - ملفات الإقلاع
يحتوي على:
- `بدء_النظام.sh`: برنامج بدء التشغيل

### `/تطبيقات` - التطبيقات والبرامج
يحتوي على:
- `إدارة_الحزم.sh`: برنامج إدارة الحزم
- `إدارة_المستخدمين.sh`: برنامج إدارة المستخدمين

### `/إعدادات` - ملفات الإعدادات
يحتوي على:
- `إعدادات_النظام.conf`: الإعدادات الرئيسية

### `/مستخدم` - مجلدات المستخدمين
مجلدات شخصية للمستخدمين

### `/مكتبات` - المكتبات المشتركة
المكتبات المستخدمة من البرامج

### `/أجهزة` - معلومات الأجهزة
معلومات العتاد والأجهزة

## استكشاف الأخطاء

### المشكلة: البرامج لا تعمل

**الحل:**
```bash
# تأكد من صلاحيات التنفيذ
chmod +x إقلاع/بدء_النظام.sh
chmod +x تطبيقات/*.sh
```

### المشكلة: عدم ظهور النص العربي بشكل صحيح

**الحل:**
```bash
# تأكد من دعم UTF-8
export LANG=ar_SA.UTF-8
export LC_ALL=ar_SA.UTF-8
```

### المشكلة: الملفات غير موجودة

**الحل:**
```bash
# تأكد من وجودك في المجلد الصحيح
pwd
# يجب أن يكون المسار: /home/username/-
```
